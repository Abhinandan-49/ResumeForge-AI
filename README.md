## ✨ Features

### 6 Professional Templates
| Template | Style | Best For |
|---|---|---|
| **Classic** | Traditional serif layout | All industries |
| **Modern** | Dark sidebar with avatar | Tech, Design |
| **Creative** | Gradient header, skill bars | Designers, Creatives |
| **Executive** | Gold accents, dark header | C-Suite, Management |
| **Tech** | Terminal/code aesthetic | Developers, Engineers |
| **Compact** | Minimal two-column | Academics, Writers |

### 🤖 Gemini AI Integration
- ✍️ **Write Professional Summary** — AI crafts a compelling intro
- 💼 **Enhance Experience Bullets** — Stronger, quantified action points
- 🛠️ **Suggest Skills** — Role-specific skill recommendations
- 🎯 **Write Career Objective** — Targeted objective statements
- ⬆️ **Improve Entire Resume** — Full review with actionable tips

### 📋 Builder Features
- **Live preview** with zoom controls (70% / 85% / 100%)
- **5 form tabs**: Personal Info, Experience, Education, Skills, Projects
- **Tag-based skills** input with keyboard shortcut (Enter to add)
- **Dynamic entries** — add/remove experience, education, projects
- **PDF Download** — pixel-perfect export via html2pdf.js
- **Sample Data** — pre-fill with example MadCoder portfolio
- **ATS-friendly** templates for all 6 layouts
- **Privacy first** — no data sent to servers, API key stored locally

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **Vanilla CSS** — Custom design system with CSS variables, animations
- **Vanilla JavaScript** — No frameworks, pure DOM manipulation
- **Gemini 1.5 Flash API** — AI content generation
- **html2pdf.js** — Client-side PDF export

## 🔑 Gemini API Key Setup

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create a free API key
3. Open the builder, paste your key in the **AI Assistant** panel
4. Click **Save** — it's stored securely in your browser's localStorage
